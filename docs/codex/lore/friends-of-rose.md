# "Friends of Rose"

!!! info "Field notes"

    **Learned:** Session 001

The call-and-response used to enter the Hellbreakers' Saringallow headquarters: knock, say "we're some friends of Rose," and the reply is "any friend of Rose can find refuge here." Ethran told you this is the expected code, and you should use it if you ever need to bring someone else to the HQ.

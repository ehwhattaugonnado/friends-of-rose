# Breachill

!!! info "Field notes"

    **First visited/mentioned:** Session 001 (Session 0 recap)

You met in Breachill before the campaign began (Session 0). It was the site of the Battle of Hellknight Hill — a significant recent event in Isger. When you left, wanted posters were up offering bounties for Andoran loyalists, Eagle Knights, and anyone opposing Chelaxian rule. Ethran Howell was recruiting there.

# How did the loyalists find the warehouse, and where did the supplies go?

!!! info "Field notes"

    **Raised:** Session 002

With Gertraud rescued, two new questions hang over the party. First, the Everbloom Imports warehouse was supposed to be secret, yet the Chelaxian loyalists knew right where to find it — nobody (not even Rhokrag) could say how. Second, there were no supplies on the ship at all; the expected crates were never there, whether intercepted in transit or never loaded. The party hasn't settled on an explanation, but the implication of an informant or a leak inside the operation is now in the air.
